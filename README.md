# 📝 Job Application Tracker

A simple frontend web application to help users manage and track job applications. Add company names, job roles, and application status, and delete entries when needed. Data is stored in the browser using Local Storage, so everything stays saved even after refreshing.

---

## 🔗 Live Demo

Check out the live project here:  
[Job Application Tracker](https://Rishika9951.github.io/Job-Tracker/)

---

## 💻 Features

- Add new job applications with company, role, and status  
- Delete job entries  
- Data persistence using Local Storage  
- Clean and responsive user interface  

---

## 🛠 Technologies Used

- **HTML5** – Page structure and form elements  
- **CSS3** – Styling and responsive layout  
- **JavaScript (Vanilla JS)** – Form handling, DOM manipulation, event handling  
- **Local Storage API** – Store and retrieve data in the browser  

---

## 📌 How It Works

1. Fill in **Company Name**, **Role**, and select **Status**.  
2. Click **Add Job** to save the entry.  
3. Jobs appear in a dynamic list below the form.  
4. Click **Delete** to remove an entry.  
5. All data is stored in Local Storage and persists even after refreshing the page.
